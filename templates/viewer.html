Metrics Reference Guide

What every metric means

Key Performance Indicators

Throughput (TPS/RPS)
Number of transactions/requests completed per second. Higher = better system capacity.
Avg Response Time
Average time (ms) for the server to respond to a request. Lower = faster.
P50 / Median
50% of requests were faster than this value. Typical user experience.
P90
90% of requests were faster than this. 10% were slower — these are the slowest typical users.
P95
95% of requests were faster. Represents near-worst-case user experience.
P99
99% of requests were faster. Worst-case experience (only 1% are slower).
Error Rate
% of requests that failed (non-200 response or timeout). Target: <1% for production.
Std Deviation
How consistent response times are. High std dev means unpredictable performance.
TC - P2P
Transaction Controller: measures total end-to-end time for all 7 USSD steps combined.
Concurrent Users
Number of virtual users running simultaneously. Simulates peak load.
Ramp-up
Time to gradually increase from 0 to full thread count. Avoids sudden server spike.
Samples
Total number of HTTP requests sent for that transaction step.

USSD P2P Transfer Flow (*1239#)

  1. Step 1 — Dial *1239#: Customer dials the USSD code, opening a session on the BTC network → menu-manager/RequestReceiver
  2. Step 2 — Input: 1359: Customer selects the P2P (Person to Person) transfer menu option
  3. Step 3 — Input: 3: Customer selects transfer type (e.g., to mobile wallet)
  4. Step 4 — Input: 1: Customer selects account type (e.g., own account)
  5. Step 5 — Receiver MSISDN: Customer enters the recipient's mobile number
  6. Step 6 — Amount: Customer enters the transfer amount in BWP
  7. Step 7 — Confirm (1): Customer confirms the transaction by entering PIN + 1 to approve

Interpreting Results — Pass / Fail Thresholds

✅ PASS Criteria
Error Rate < 1%
Avg Response < 3,000 ms
P90 < 5,000 ms
Throughput stable (not dropping)
No Out-of-Memory errors
❌ FAIL Criteria
Error Rate > 5%
P99 > 10,000 ms
Throughput degrading over time
Connection timeouts / Socket errors
Server 5xx errors
