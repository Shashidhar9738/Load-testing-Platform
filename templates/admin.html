How to Run a Test
  1. Select a JMX test plan from the dropdown
  2. Set Threads = number of concurrent virtual users
  3. Set Duration in seconds (e.g. 600 = 10 min)
  4. Set Ramp-up — time to reach full thread count
  5. Optionally name the output file (auto-named if blank)
  6. Click ▶ Start Test
  7. Watch the live log below
  8. After completion, go to Reports to view results
📌 JMX Parameters Used
test.duration → Duration in seconds
p2p.threads → Thread count
p2p.rampup → Ramp-up period
Extra params override defaults in the JMX
